#
```data |address|sharcode|companyName|covered|device_num|employee_num|fixed_assets|obtainYear|rent|yTurnover|area_id|companyTypes|mianIndustrys| ```
#
```参数说明 | 经营地址|总资产|企业名称|占地面积|设备数量|员工数量|固定资产|实际控制人从业年限| 租用|年营业额|经营地址区县|公司类型：11,22,33| 主营行业：11,22,33 ```
